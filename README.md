# Opspace

A personal operating system for solo builders.

Opspace helps you capture ideas, manage projects, execute tasks, and review your work through a structured system.

## Status

Pre-MVP

## Core Principles

- Fast capture over complex input
- Opinionated system over flexibility
- Execution and review over storage
- Local-first where possible

## MVP Scope

- Projects
- Tasks
- Ideas
- Notes
- Quick capture
- Daily view
- Weekly review
- Basic insights

## Tech Stack

- Backend: Java (Spring Boot)
- Frontend: React / Next.js
- Database: SQLite (initial)
